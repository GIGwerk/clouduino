# Clouduino

![Logo](logo.png)

Clouduino is a fully open-source, WiFi mesh-enabled embedded development platform.

## Specs

- CPU: 160 MHz
- L106 32-bit RISC Tensilica Xtensa Diamond Standard 106Micro
- Flash Memory: 4MB
- 16 GPIO pins
- SPI
- I2C support
- 12-bit SAR ADC support
- IEEE 802.11 b/g/n WiFi (w/ WEP, WPA, WPA2)
